An identity and a first run worth sharing.

Three monochrome logo directions plus the six-screen first-run flow that fetches the speech model on demand. Everything below stays in the black-and-white language; the soft-gold (#E8B478) appears only as the download / processing accent, the same signal the listening capsule uses while transcribing.

Logo concepts

Each mark is shown at its hero size (88px), at tray size (16px, scaled up here) to prove it survives, and in the transient gold processing state. All are filled silhouettes so they stay solid at any size; the detail is negative space, not added strokes.

Recommended · hero

A — The Waveform Capsule

A vertical pill with three bars cut through it in negative space — a frozen waveform. It is literally the app's own listening capsule, so logo, tray icon and on-screen pill become one idea.

C — The Negative-Space Mic Pill

A rounded tile (echoing the 8px window radius) with a minimal microphone cut out of it. Keeps the familiar mic cue, but ownable because the mic is absence inside your capsule.

First-run flow · six screens

Each screen has one primary action, immediate-apply, SettingsCard styling. The user reaches a working verbatim-dictation state by screen 5; AI is a clearly optional detour, never required. Pattern distilled from LM Studio, MacWhisper, SuperWhisper and Ollama.
